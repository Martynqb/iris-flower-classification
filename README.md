# Iris Flower Classification 🌸

## 📌 Project Overview
This project is part of my Data Science Internship at Oasis Infobyte.  
The goal is to classify iris flowers into three species:
- Setosa
- Versicolor
- Virginica

## ⚙️ Technologies Used
- Python
- Scikit-learn
- NumPy

## 🤖 Model Used
K-Nearest Neighbors (KNN)

## 📊 Steps Involved
1. Load Dataset
2. Split Data into Training and Testing
3. Train Model
4. Predict Output
5. Calculate Accuracy

## ✅ Result
The model achieved good accuracy in predicting the flower species.
